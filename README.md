# Pharmacy Management System (JavaFX)

A full desktop application built using Java and JavaFX to manage pharmacy operations.

## Features
- Role-based system (Admin, Cashier, Customer)
- Product & supplier management
- Order handling and ratings
- Advanced reports (revenue, best sellers)
- Date-based filtering
- JavaFX GUI with TableView & popups

## Admin Capabilities
- Manage users (add, edit, remove)
- Manage products and suppliers
- View and filter orders
- Generate reports (supplier, product, cashier, customer)

## Technologies
- Java
- JavaFX
- Object-Oriented Programming (OOP)
- Serialization
